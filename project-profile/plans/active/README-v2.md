# Active Plans

Store one Markdown file per active task.

Suggested template:

```md
# Task Name

## Goal

One paragraph on the intended outcome.

## Scope

- in scope
- out of scope

## Steps

1. inspect current state
2. implement smallest correct change
3. validate
4. document follow-up if needed

## Decision Log

- Decision: ...
- Why: ...

## Validation

- Ran: ...
- Result: ...

## Follow-up

- ...
```

Move completed plans into `plans/completed/`.
