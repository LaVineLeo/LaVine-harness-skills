# QUALITY_SCORE

Use this rough rubric when judging changes in this repo.

## Score Dimensions

- profile clarity: can a human fill the template without guessing?
- skill clarity: does the skill know when and how to use the profile?
- scaffold stability: does the template create a clean first draft?
- boundary discipline: are generic and project-specific layers kept separate?
- repo legibility: can the next agent quickly understand the generated structure?
