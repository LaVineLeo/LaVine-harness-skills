# PRODUCT_SENSE

The main product in this repo is not code generation by itself. The product is a reusable Harness setup path.

## Primary User Need

A team should be able to say:

- here is our project purpose
- here are our repos
- here is our workflow

and then get a useful first Harness without rewriting everything from scratch.

## Highest-Leverage Output

The highest-leverage output is a reusable skill plus a clear project profile template.
