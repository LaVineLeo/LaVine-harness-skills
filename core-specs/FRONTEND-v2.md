# FRONTEND

This repo is not primarily a frontend project, but the generated Harness may be used in frontend-heavy monorepos.

## Rule

If a target project has important UI work, encode the real frontend boundaries, quality bars, and review rules in the generated repo instead of keeping them implicit.

## What Not To Do

Do not hard-code one frontend stack into the generic Harness unless the profile explicitly requires it.
